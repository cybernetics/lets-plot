lets_plot.geo_data.distance¶

lets_plot.geo_data.distance(lon0, lat0, lon1, lat1, units='km')¶

Calculate the distance between two points. Returns result in kilometers or miles.

distance(lon0, lat0, lon1, lat1, units)

Parameters

  • lon0 (number) – Longitude coordinate of the first point.

  • lat0 (number) – Latitude coordinate of the first point.

  • lon1 (number) – Longitude coordinate of the second point.

  • lat1 (number) – Latitude coordinate of the second point.

  • units ([string | None]) – The units in which the result will be obtained. There are shorthands for values: ‘mi’(miles), ‘km’(kilometers). Default is kilometers.

Returns

object

Return type

float

Note

distance() calculates the distance between two points.

Examples

>>> from lets_plot.geo_data import *
>>> dist = distance(-99.25061, 31.25044, -105.50083, 39.00027)
>>> dist
